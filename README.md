
# Auth Api

A set of Api's that can be used for user signup and login.Implemented private routes and jwt validation. 


## Tech Stack

**Server:** Node, Express, MongoDB


## Features

- Login/Signup
- Jwt authentication
- Validation using Joi Library
- Private route
- Password encryption using bcrypt


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

#### Server side env

`PORT`

`JWT_SECRET`

`DB_URL`

## Run Locally

Clone the project

```bash
  git clone (https://github.com/rahul-6-ramachandran/skygoal-Task.git)
```

Go to the server directory

```bash
  cd <directory_name>
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index
```
